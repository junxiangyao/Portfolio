CHALLENGE

We were challenged to design a white-label platform for ai.Retial with straightforward user experience and enable users to access data and make decisions with ease.



OmniChannel Analytics

This project is building a working prototype using Omnichannel Analytics to understand customers’ existing preferred path to purchase, forecast the potential combination of channels that drive sales & thereby use those insights to allocate marketing budgets in the right set of channels to increase their top line and marketing ROI.


Research & Ideation

Collaborating with our client, we started by creating job stories for the major persona, the marketing managers. From there, we brainstormed the features desired and the use cases to guide the next step.

Because this feature is more on presenting data and insights instead of interactions with multiple touchpoints, it has a relatively short customer journey. So I created a conceptual model of this product to serve as the skeleton, which is more appropriate than traditional information architecture.

Iteration

After the ideation phase, I started the wireframe sprints and visual design sprints and iterated on them to incorporate the feedback from the client. In the end, the deliverable includes the research and ideation process file, wireframe file, visual design file including both desktop and mobile version, and an overarching design system that is shared by all the features and modules.

Store Resources Management with Curbside Delivery

As another feature I worked on for ai.RETAIL, this project is building a working prototype for store managers to manage store resources and with the curbside delivery system.


Research & Ideation

Similar to Omnichannel Analytics, we started by creating job stories not only for our major persona, the store manager, but also for the booth receptionist and the customers, as they are all critical in the system I was desinging.

Because this feature is a complicated system that involves multiple roles and touch points, I created a IA and Service Blueprint for it, as well as the user flows for all three personas to better understand the experience.

Iteration

I started the wireframe sprints followed by the visual design sprints. The deliverable for this feature includes the research and ideation process file, wireframe file, visual design file including the dashboard design as well as the booth receptionist's dashboard, kiosk and mobile UI for better storytelling.
